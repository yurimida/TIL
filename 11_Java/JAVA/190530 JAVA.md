* 변수는 데이터를 보관하는 장소(memory)

* 변수는 사용하기 전에 선언되어야 한다. 변수의 선언이란 "이름"과 "타입"을 정해주는 것

* 변수는 "적용 범위(scope)"를 가짐

​	메서드(블록)내부에 선언된 변수는 그 메서드(블록) 내부에서만 사용가능

​	메서드 외부에 (클래스 내부) 선언된 변수는 클래스 내에서 사용가능
